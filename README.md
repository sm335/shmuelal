# Shmuelal

Final project submitted as part of my Practical Engineer diploma program.

Built using HTML, CSS, and JavaScript, this project showcases a complete and functional website. As part of the project evaluation, students were given on-the-spot tasks to demonstrate problem-solving and the ability to work under pressure.

My assigned task was implementing a functional click-to-expand image feature, allowing images to expand and collapse dynamically.

Final project grade: 100/100.
